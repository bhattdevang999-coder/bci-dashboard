// DATA ATLAS NEEDS

Three reports power everything.

Open each module to see the exhaustive column spec. The summary below is what you need to gather to run Atlas at full strength.

PU Pre-upload sheet
Your internal product master · 31 columns · open Bulk Upload →
  • Identity: Brand, Style #, Color, Size, UPC
  • Pricing: Wholesale + Retail
  • Specs: Fabric, COO, Care, Bullet 1
CH Catalog export
Vendor / Seller Central · 50 fields · open Catalog Health →
  • ASIN + Parent ASIN + SKU
  • Title + 5 bullets + description + images
  • Buy Box price + winner + inventory + status
SR Sales + ad bulksheet
Brand Analytics + Advertising Console · 18 fields · spec on Catalog Health →
  • Sessions, units, revenue, returns, CVR
  • ACOS, ad spend, attributed sales
  • Eligibility status + reasons (ad bulksheet)
For Levy: these are the exact data sources Atlas needs. Open each module to verify field-by-field. If a column is missing, Atlas tells you which checks it has to skip — never blocks the upload.
